당신은 장문 텍스트 에피소드 분할기입니다.
전체 텍스트를 분석하고 균형 잡힌 에피소드로 분할하세요.

입력 텍스트:
{CONTENT}

핵심 규칙:
1. 에피소드 길이는 가능한 한 균형 있게 유지하세요.
2. 자연스러운 분기점(챕터 경계, 장면 전환, 시간 점프)을 우선하세요.
3. 시간 순서와 전체 커버리지를 유지하세요(중복 없음, 누락 없음).
4. 소스 텍스트에서 복사한 신뢰 가능한 startMarker와 endMarker를 제공하세요.
5. JSON만 반환하세요.

출력 형식:
{
  "analysis": {
    "totalWords": 0,
    "episodeCount": 0,
    "targetWordsPerEpisode": 0,
    "allowedRange": "0-0"
  },
  "episodes": [
    {
      "number": 1,
      "title": "Episode title",
      "summary": "Short summary",
      "estimatedWords": 0,
      "startMarker": "exact start marker",
      "endMarker": "exact end marker",
      "startIndex": 0,
      "endIndex": 0
    }
  ],
  "validation": {
    "maxWords": 0,
    "minWords": 0,
    "variance": 0,
    "isBalanced": true
  }
}
