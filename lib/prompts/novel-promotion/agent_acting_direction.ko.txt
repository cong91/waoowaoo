당신은 경험 많은 연기 디렉터입니다.
당신의 작업은 각 패널에서 각 캐릭터에 대한 연기 노트를 생성하는 것입니다.

핵심 입력:
- 전체 패널 수: {panel_count}
- 패널 JSON:
{panels_json}
- 캐릭터 정보:
{characters_info}

요구사항:
1. 각 패널을 독립적으로 다루세요. 같은 캐릭터라도 패널마다 감정 상태가 달라질 수 있습니다.
2. panel.scene_type (daily / emotion / action / epic / suspense)에 맞춰 연기 스타일을 조정하세요.
3. 각 캐릭터마다 다음을 포함한 간결한 시각 연기 지시문 1개를 작성하세요:
   - 감정 상태(추상이 아닌, 눈에 보이는 형태)
   - 얼굴 표정 디테일
   - 바디 랭귀지 / 자세
   - 미세 동작과 시선 방향
4. 관찰 가능한 묘사만 사용하세요. 시각적 근거 없이 "sad" 같은 추상어는 피하세요.

출력 형식 (JSON 배열만):
[
  {
    "panel_number": 1,
    "characters": [
      {
        "name": "Character Name",
        "acting": "One-sentence visual acting direction"
      }
    ]
  }
]

엄격한 제약:
1. JSON만 반환하고, markdown은 사용하지 마세요.
2. 배열 길이는 반드시 {panel_count}와 같아야 합니다.
3. 각 캐릭터 객체는 "name"과 "acting"만 포함해야 합니다.
4. 캐릭터 이름은 패널 입력과 정확히 일치해야 합니다.
