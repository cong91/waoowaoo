# 레퍼런스 이미지 → 캐릭터 시트 프롬프트 (img2img)

제공된 레퍼런스 이미지를 사용해 얼굴 특징, 헤어스타일, 체형, 의상 구조를 추출하세요.

## 스타일 일관성
원본 비주얼 스타일을 유지하세요:
- 실사 사진 -> 리얼리스틱 스타일
- 애니메이션 -> 애니 스타일
- 사실적 일러스트 -> 사실적 일러스트 스타일
- 그 외 모든 스타일 -> 해당 스타일을 그대로 보존

## 생성 규칙
1. 원본 이미지의 색감 왜곡과 조명 결함은 무시하세요
2. 깨끗하고 부드러운 스튜디오 조명을 사용하세요
3. 자연스럽고 미적으로 올바른 신체 비율을 유지하세요
4. 블러, 노이즈, 압축 아티팩트, 결함을 복제하지 마세요
5. 출력은 선명하고 또렷한 프로덕션 품질이어야 합니다
6. 캐릭터 표정은 중립적이고 차분하며, 카메라를 바라봐야 합니다

## 누락 부위 보완
반신 또는 신체 일부만 보일 경우, 가려진 부분을 일관성 있게 추론하여 보완하세요:
- 어울리는 하의 의상을 추론
- 적절한 신발을 추론
- 누락된 손/팔을 자연스럽게 추론
