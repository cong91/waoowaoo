# 캐릭터 이미지 설명 프롬프트

제공된 캐릭터 이미지를 분석하고, 이미지 생성에 사용할 수 있는 상세한 영어 시각 설명 1개를 작성하세요.

## 필수 포함 내용
다음 항목을 모두 포함하세요:
1. 성별 및 대략적인 연령대
2. 헤어스타일과 머리색
3. 얼굴형과 얼굴 특징
4. 체형과 실루엣
5. 의상 스타일과 의상 디테일
6. 액세서리와 시그니처 디테일
7. 전체 스타일 키워드

## 누락 부위 보완
이미지에 신체 일부만 보일 경우, 누락된 부분을 일관성 있게 추론해 보완하세요.
- 하반신이 누락된 경우: 어울리는 바지/치마 스타일을 추론
- 신발이 누락된 경우: 의상에 맞는 신발을 추론
- 액세서리 디테일이 누락된 경우: 합리적인 액세서리를 몇 가지 추론

## 금지 내용
다음은 언급하지 마세요:
- 피부색
- 눈 색
- 표정
- 포즈 또는 동작
- 배경

## 출력 형식
영문 일반 문단 1개만 반환하세요 (약 120~220단어).
마크다운, 불릿, 제목, JSON은 반환하지 마세요.
