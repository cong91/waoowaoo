# Task 6 — Add first-use guidance for manga quick actions

## Problem
Quick actions are powerful but currently under-explained for first-time users.

## Goal
Help users understand what Add / Duplicate / Split / Merge / Reorder actually do and when to use them.

## Acceptance criteria
- each quick action has plain-language guidance
- first-time user can infer safe/default action without trial-and-error
- action descriptions focus on user outcomes, not internal terms
- optional: differentiate beginner-safe actions vs advanced restructuring actions

## Suggested copy direction
- Add = create a new panel after the current point
- Duplicate = reuse a similar panel quickly
- Split = break one long panel into two beats
- Merge = combine two short panels into one
- Reorder = change reading flow order
