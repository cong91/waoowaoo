# Task 4 — Reposition or collapse demo/business framing in manga flow

## Problem
The `Demo flow setup` section (Launch Teaser / Brand Story / Product Explainer) distracts from the beginner manga creation journey.

## Goal
Reduce off-path business/demo framing in the main manga beginner flow.

## Acceptance criteria
- `Demo flow setup` is not a primary interruption in the default beginner manga path
- section is either collapsed, moved lower, or moved into advanced/business context
- manga-first creation remains the dominant mental model on the page
- users no longer confuse the manga flow with ad/demo tooling
